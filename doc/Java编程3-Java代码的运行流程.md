## 3. 代码安全机制

Java通过下面方法确保其安全性：

1. 将代码编译成class文件=》编译检查；
2. 将本地类和网络资源类分离=》类装载检查；
3. 类装载后进行字节码校验；
4. 网络运行程序时使沙箱。
